## Description

프로젝트 바플입니다. 

바플은 바디 플래너의 약자로 효율적으로 운동 기록을 도와주는 서비스입니다.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
